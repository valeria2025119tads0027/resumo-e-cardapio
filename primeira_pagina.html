Capítulo 2

Estruturando Páginas Web com HTML

2.1 Introdução

O HTML (Hypertext Markup Language) é a linguagem fundamental para definir a estrutura e a semântica (o significado) dos componentes de uma página na web. Ele funciona como uma base onde você destaca textos, insere imagens e cria tabelas, de forma semelhante a programas como o Microsoft Word.

2.1.1 Versões

O HTML é a linguagem de marcação que define a estrutura e a semântica das páginas web. O documento destaca que a linguagem não é estática:
Evolução: Passou pelo HTML 4.0, pelo rigoroso XHTML e culminou no HTML5 em 2014, que trouxe suporte nativo para áudio, vídeo e novas formas de interação.
Living Standard: Atualmente, o HTML não possui mais números de versão (como "6" ou "7"). Ele é tratado como um "Padrão Vivo", atualizado continuamente pelo WHATWG para refletir as necessidades modernas da web.

2.1.2 Elementos e Atributos

Este tópico explica a gramática básica da linguagem:
Elementos: São os componentes da página (parágrafos, botões, etc.). A maioria possui uma tag de abertura e uma tag de fechamento.
Atributos: São propriedades inseridas dentro da tag de abertura para modificar seu comportamento ou fornecer dados extras (ex: href para endereços de links ou src para o caminho de uma imagem). Seguem sempre o formato nome="valor".

2.1.3 Estrutura de Documentos HTML

Um documento HTML é um arquivo de texto simples com extensão .html. Ele deve seguir uma hierarquia obrigatória:
<!DOCTYPE html>: Informa ao navegador que o arquivo usa a versão mais recente do HTML.
<html>: O elemento "pai" que envolve todo o conteúdo.
<head>: Contém metadados (informações técnicas), como a codificação de caracteres (UTF-8) e o <title> (nome que aparece na aba do navegador).
<body>: Onde reside tudo o que o usuário realmente vê na tela.

2.2.1 Elementos de Texto e Formatação

Este código detalha como organizar o conteúdo textual:
Hierarquia de Títulos: Vai de <h1> (título principal) até <h6> (subtítulo de menor nível).
Parágrafos e Quebras: Texto comum é colocado em <p>. Para pular linha sem criar um novo parágrafo, usa-se <br />.
Semântica de Destaque: <strong> ou <b> para negrito; <em> ou <i> para itálico.
Outros elementos: <sup> (sobrescrito), <sub> (subscrito) e <blockquote> para blocos de citações recuados.

2.2.2 Criação de Listas

Existem três tipos de listas em HTML:
Listas Ordenadas (<ol>): Itens numerados automaticamente (1, 2, 3...).
Listas Não Ordenadas (<ul>): Itens marcados com pontos ou símbolos, onde a ordem não importa.
Listas de Definição (<dl>): Usadas para glossários, relacionando um termo (<dt>) a uma descrição (<dd>).
Aninhamento: É possível colocar uma lista dentro de outra para criar subníveis.

2.2.3 Definindo Links

O recurso que conecta a web. Criados com a tag <a>:
Externos: Apontam para endereços completos (URLs) de outros sites.
Relativos: Apontam para arquivos dentro do seu próprio projeto ou pasta.
Internos: Permitem criar links que levam o usuário para uma parte específica da mesma página usando o atributo id.

2.2.4 Incluindo Imagens

Para inserir imagens, utiliza-se a tag <img>:
Atributos Essenciais: src (onde a imagem está guardada) e alt (texto que descreve a imagem para deficientes visuais ou caso o arquivo não carregue).
Legendas: O HTML5 introduziu o par <figure> (para envolver a imagem) e <figcaption> (para o texto da legenda logo abaixo).

2.2.5 Tabelas

Usadas para dados tabulares (não para layout):
Estrutura básica: <table> contém linhas (<tr>), que por sua vez contêm células de dados (<td>) ou células de cabeçalho (<th>). Organização Avançada: Uso de <thead>, <tbody> e <tfoot> para separar o topo, o conteúdo e o rodapé da tabela. Mesclagem: Atributos colspan (mesclar colunas) e rowspan (mesclar linhas).

2.2.6 Formulários

Permitem a interação direta do usuário:
Tag <form>: Envolve todos os campos e define para onde os dados serão enviados.
Campos de Entrada (<input>): Podem ser do tipo text, password (oculta caracteres), checkbox (múltipla escolha) ou radio (escolha única).
Rótulos: A tag <label> é fundamental para acessibilidade, ligando um texto ao seu respectivo campo.
Envio: Botões do tipo submit disparam o envio das informações.
